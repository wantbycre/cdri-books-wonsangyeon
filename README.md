# 씨디알아이(CDRI) BOOKS

씨디알아이 프론트 사전과제 kakao api 연결 books

## stack

-   nextjs
-   typescript
-   tailwind
-   react-query
-   jotai
-   shadcn

## start

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
